# Fundimart Backend

Backend API for Fundimart, a construction materials marketplace platform.

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Authentication
- Swagger API Docs
- Joi Validation

## Features

- User registration and login
- Role-based access control
- Product management
- Order creation with stock deduction
- Payment initiation and confirmation
- Admin order management
- PostgreSQL persistence

## Project Structure

src/
- config/
- controllers/
- middlewares/
- routes/
- services/
- utils/
- validators/

prisma/
- schema.prisma
